### Not a video-person? Follow this simple guide:

Before you continue with this tutorial, there are a couple of requirements to
set your custom server thumbnail.

1. Your server icon must be a **64x64** size image
2. Your server icon has to be in the **.PNG** format
3. Your server icon is named `server-icon`

### All requirements met? Read below:

1. Login to [Seedloaf](https://seedloaf.com)
2. Access your world's File Manager
3. Upload the **.PNG** file into the root folder
4. Restart your server

After the restart is complete, your server icon should show to everyone and look
